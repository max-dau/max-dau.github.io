About

So now, I have my YouTube channel and have started to really enjoy making videos, streaming, graphic designing, etc.. Now, I had the idea of building a website for my channel. At first, I wanted to build it with Squarespace or some similar service, but I decided that I could start doing it from scratch with HTML and graphic design, because I in general don't like using templates, especially because I learned to graphic design and learned the very basics of HTML and webdesign and so I have the ability to make a site which I can call my own, unique site, also when it is far away from the perfect webpage, but I'm quite surprised with how the page turned out. So this site will probably never be totally finished, I (at least hope) that I will keep working on it and improving it in at least the near future.
